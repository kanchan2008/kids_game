# Tick Tock Time Game

A small, kid-friendly analog clock game that runs as a static website and is ready for GitHub Pages.

## How it works

- The game shows a random analog clock time in 5-minute steps.
- Kids choose the matching digital time from four answer buttons.
- The game validates the answer, shows a friendly correct or wrong message, and immediately moves to the next random clock.
- Correct and wrong answer counters help kids track their practice.

## Deploy on GitHub Pages

1. Push these files to a GitHub repository branch, such as `clock-quiz-github-pages`.
2. Open the repository on GitHub.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the branch and the repository root folder, then save.
6. Open the GitHub Pages URL after the deployment finishes.
